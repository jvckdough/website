# Academic Personal Website — John Allerton Doughty

A clean, responsive academic profile website built with plain HTML, CSS, and vanilla JavaScript.

## Files

- `index.html` — site content and structure
- `styles.css` — visual styling and responsive layout
- `script.js` — mobile nav toggle and footer year
- `assets/` — place static files here (e.g., CV PDF)

## CV Placeholder

The **Download CV** button currently points to:

`assets/John_Allerton_Doughty_CV.pdf`

Add your CV PDF at that path to enable downloading.

## Run Locally

From this project directory:

```bash
cd /Users/jackdoughty/.openclaw/workspace/academic-site
python3 -m http.server 8000
```

Then open:

`http://localhost:8000`
